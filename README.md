# ஸ்ரீ செல்வ விநாயகர் இளைஞர்கள் (SVK BOYS) - வலைத்தளம்

## 🎉 வலைத்தளம் வரவேற்பு

இந்த வலைத்தளம் ஸ்ரீ செல்வ விநாயகர் இளைஞர்கள் நற்பணி மன்றத்தை பிரதிநிதித்துவம் செய்கிறது. இது புகைப்படங்கள், நிகழ்வுகள் மற்றும் சமூக நிகழ்ச்சிகளை வர்ணிக்கிறது.

---

## 📁 கோப்பு கட்டமைப்பு

```
svk web/
├── index.html          # முக்கிய HTML கோப்பு
├── style.css           # முழு CSS ஸ்டைல்கள்
├── script.js           # ஜாவாஸ்க்ரிப்ட் செயல்பாடுகள்
└── images/
    ├── hero-bg.jpg     # முகப்பு பொதுவு படம்
    └── SVK BOYS/       # புகைப்படங்கள் மற்றும் வீடியோக்கள் ஃபோல்டர்
        ├── klp1_*.* (பொங்கல் விளையாட்டு)
        ├── klp2_*.* (அழகுநாச்சியம்மன் - பூச்சொரிதல்)
        ├── klp3_*.* (17-வது மண்டகப்படி)
        ├── klp4_*.* (கோடை நிகழ்ச்சி)
        └── klp5_*.* (விநாயகர் சதுர்த்தி)
```

## 🎨 வலைத்தளத்தின் பிரதான பகுதிகள்

### 1️⃣ **முகப்பு (Hero Section)**
- மகத்தான பொதுவு படம் `hero-bg.jpg`
- வணக்கம் செய்தி மற்றும் அழைப்பு பொத்தான்கள்

### 2️⃣ **நிகழ்வுகள் பகுதி**
- 5 முக்கிய நிகழ்வுகள் கার্டுகளாக காட்டப்படும்
- ஒவ்வொன்றும் மாதம் மற்றும் விளக்கம் கொண்டுள்ளன

### 3️⃣ **புகைப்பட கூடம் (Gallery)**
- **நிகழ்வு வடிகட்டி**: அனைத்தும், klp1-klp5
- **ஆண்டு வடிகட்டி**: 2022-2026
- **Lightbox முன்னோட்டம்**: முழு பதிப்பு பெரிய பார்வைக்கு
- **வீடியோ ஆதரணை**: MP4 வடிவம்
- **எழிப்பு প্রভாவம்**: மிளிரும் அ্যানிমேஷன்

### 4️⃣ **சாட்சிப்பத்திரங்கள்**
- ஊர் மக்களின் நன்மொழிகள்
- 3 மொழிப்பெயர்ப்புக்கள் உள்ளன

### 5️⃣ **தொடர்பு பகுதி**
- தொடர்பு பத்திவளை மாதிரி
- அலுவலக முகவரி மற்றும் சமூக ஐகன்கள்

### 6️⃣ **அடிக்குறிப்பு (Footer)**
- விரைவு இணைப்புகள்
- முக்கிய விழாக்கள் பட்டியல்
- நির்ணய உரிமைகள் சாதனை
